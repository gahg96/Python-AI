📋 系统概述

Gamium Finance AI 是一个基于AI的智能金融决策支持系统，通过数据蒸馏、世界模型预测、风险控制压力测试、多模型对抗评测等先进技术，为金融机构提供全方位的客户风险评估、策略优化和决策支持。

🚀 核心功能模块

🔬 数据蒸馏 (Data Distillation) 核心功能

将历史数据转化为可调用的"世界模型"，用于预测客户未来行为。

  • 自动生成符合真实分布的客户数据
  • 支持个人客户和企业客户两种类型
  • 个人客户类型：工薪族、专业人士、创业者、投资者、退休人员、学生
  • 企业客户类型：初创企业、科技初创、制造企业、贸易公司、服务企业
  • 智能预测客户违约概率、流失概率、预期LTV
  • 风险评分计算和详细解释
  • 历史贷款记录下钻查看
  • 逾期贷款详细信息展示

👤 客户预测 (Customer Prediction) 核心功能

基于AI模型对客户进行全面的风险评估和未来预测。

  • 支持个人/企业客户类型切换
  • 自动生成或手动输入客户信息
  • 预测违约概率（可下钻查看计算过程）
  • 预测流失概率（可下钻查看计算过程）
  • 计算预期LTV（可下钻查看计算过程）
  • 风险评分（可下钻查看计算规则）
  • 历史贷款记录查看
  • 逾期贷款详情展示
  • 风险因子贡献度分析
  • 决策规则链展示

⚠️ 风控压力测试 (Risk Control Stress Test) 新功能

在虚拟环境中注入各类"黑天鹅"事件，测试现有策略的韧性和恢复能力。

  • 支持多种黑天鹅事件类型：金融危机、疫情冲击、行业暴雷、利率冲击、失业率飙升
  • 可调节事件强度：轻度、中度、重度、极端
  • 可设置事件持续时间（月）
  • 测试客户配置：客户类型、行业、城市层级、数量
  • 贷款条件配置：金额、利率、期限
  • 敏感性分析：GDP、失业率、利率、通胀率
  • 韧性评估：违约率、利润回撤、恢复时间、资本充足率
  • 详细计算步骤可下钻查看
  • 客户详情统计分析
  • 自动生成应急预案

🤖 多模型对抗评测 (Multi-Model Adversarial Evaluation) 新功能

评估不同大语言模型和智能代理在金融场景下的表现。

  • 支持18+主流模型：OpenAI、Anthropic、Google、阿里云、DeepSeek、Kimi、GLM、Ollama等
  • 模型分类筛选：全部、OpenAI、Anthropic、本地模型、RAG应用
  • 模型搜索功能
  • 全选/取消全选功能
  • 切换分类时保留选中状态
  • 12个专业评测维度：正确率、幻觉率、合规率、专业性、响应质量、一致性、业务适配度、延时、错误率、稳定性、性能评分、综合评分
  • 5个预定义测试用例：贷款审批、财富管理、信用风险评估、客户投诉、财务规划
  • 多轮对话测试（每用例5-8轮）
  • 完整对话历史记录
  • 评分依据可查看（用于合规审计）
  • 测试用例详情查看
  • 对话内容保存为标准话术模板
  • 支持导出JSON、TXT、CSV格式

📖 典型使用流程

1. 客户风险评估流程

  1. 进入"客户预测"页面
  2. 选择客户类型（个人/企业）
  3. 填写或自动生成客户信息
  4. 设置贷款条件
  5. 点击"生成客户并预测"
  6. 查看预测结果（违约概率、流失概率、LTV、风险评分）
  7. 点击各项指标查看详细计算过程
  8. 查看历史贷款记录和逾期详情

2. 风控压力测试流程

  1. 进入"风控压测"页面
  2. 选择黑天鹅事件类型和强度
  3. 配置测试客户参数
  4. 设置贷款条件
  5. 配置敏感性分析维度
  6. 设置韧性评估参数
  7. 点击"开始压力测试"
  8. 查看测试结果和详细分析
  9. 点击各项指标查看计算说明
  10. 查看应急预案建议

3. 模型评测流程

  1. 进入"模型评测"页面
  2. 选择要评测的模型（可多选）
  3. 选择测试用例（可多选）
  4. 点击"开始评测"
  5. 查看评测结果和排名
  6. 查看详细对话内容
  7. 查看评分依据
  8. 保存对话为话术模板
  9. 导出评测数据

4. AI Hub 管理流程

  1. 进入"AI Hub"页面
  2. 模型管理：点击左侧模型列表选择模型，在右侧配置API Key、限额、熔断等
  3. 敏感词管理：添加/删除敏感词
  4. 权限控制：管理用户和角色
  5. 使用统计：查看调用统计、成本统计、异常监控，点击"查看详情"进行多维度下钻
